# Каталог пользователей с пагинацией и поиском по имени, реализованный на React + TypeScript + Tailwind CSS, с использованием публичного API dummyjson.com

Функционал:

- Отображение списка пользователей с основной информацией:
    - Имя, фамилия
    - Email
    - Аватар
- Поиск по имени (по первой букве) с debounce 300ms
- Пагинация с кнопками ← и → и выбором страницы
- Hover эффекты и курсор pointer для кнопок и карточек
- Адаптивная сетка пользователей (1–3 колонки в зависимости от ширины экрана)

Технологии:

- React 18 + TypeScript
- Tailwind CSS
- Vite (сборка и dev-сервер)
- Публичный API: https://dummyjson.com/users

Запуск:
```bash
npm install
npm run dev
